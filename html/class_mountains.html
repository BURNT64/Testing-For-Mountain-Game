Detailed Description

A class for working with a set of mountain names and ranges.

This class represents a set of mountain names and ranges that can be read from a set of files, and provides methods for selecting a random mountain and checking whether a given mountain belongs to a given range.

This constructor takes a vector of filenames containing mountain and range data, reads the data from each file, and stores the mountain names and ranges internally.

Time complexity: O(N*M), where N is the number of files and M is the number of lines in each file.
